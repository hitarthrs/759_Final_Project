## 759_Final_Project
Final Project for CS759 - Fall 2024

#### Getting started on Linux
- Install Docker
- Build fractal-image docker image from within project dir
    `sudo docker build -t fractal-image .`
- Launch image
    `./openbuildpack.sh`
- Build
    - `mkdir build`
    - `cd build`
    - `cmake ..`
    - `make`
- Run
    `./fractal`
